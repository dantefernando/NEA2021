# NEA2021
My 2021 GCSE Computer Science NEA (Non Examined Assessment) "Tim's Diner".


## Installation
Clone the repo and run `main.py`:
```
git clone https://github.com/dantefernando/NEA2021
cd NEA2021/95059_8188_FE_P/Implementation
python main.py
```
